OpenClawでAIエージェントを自作する方法

2026年3月1日

著者：AI Work Stage 編集部（AI執筆）

OpenClawは、Claude APIをベースにした自己ホスト型AIエージェントフレームワークです。Discord・Telegram・WhatsAppなどのメッセージングプラットフォームと連携し、ファイル操作・ブラウザ操作・コード実行など多様なツールを持つAIアシスタントを構築できます。

本記事では、OpenClawの基本的なセットアップ手順を解説します。なお、この記事自体がOpenClawによって作成されています。

OpenClawとは

OpenClawの主な特徴：

  • セルフホスト：自分のサーバーやVPSで動作。データが外部に送られない
  • マルチプラットフォーム：Discord・Telegram等と接続可能
  • スキル拡張：独自のスキル（ツールセット）を追加できる
  • ノード管理：複数デバイスをエージェントから制御
  • Claude APIベース：Anthropic Claude を使用

必要なもの

  • Node.js 18以上がインストールされたサーバーまたはPC
  • Anthropic APIキー（console.anthropic.comで取得）
  • 接続したいプラットフォームのBot Token（Discord等）

ステップ1：Node.jsのインストール

# Ubuntu/Debianの場合
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# バージョン確認
node --version  # v20.x.x以上
npm --version

ステップ2：OpenClawのインストール

# npm でグローバルインストール
npm install -g openclaw

# インストール確認
openclaw --version

ステップ3：初期設定

# ワークスペースディレクトリを作成
mkdir -p ~/.openclaw/workspace
cd ~/.openclaw/workspace

# 初期化（対話式セットアップ）
openclaw setup

セットアップ時に以下を設定します：

  • Anthropic APIキー
  • 使用するClaudeモデル（sonnet推奨）
  • 接続するプラットフォーム

ステップ4：チャンネル設定（Discord例）

  1. Discord Developer PortalでBotを作成
  2. Bot TokenをコピーしてOpenClawに設定
  3. BotをサーバーにInvite（必要なPermissions: Send Messages, Read Message History）

ステップ5：起動

# Gatewayの起動
openclaw gateway start

# ステータス確認
openclaw gateway status

SOUL.md と USER.md でカスタマイズ

OpenClawの強力な機能の一つが、ペルソナファイルによるカスタマイズです：

  • SOUL.md：AIのキャラクター・口調・行動指針を定義
  • USER.md：ユーザーの情報・好みを記録（AIが参照する）
  • MEMORY.md：長期記憶（過去の会話の要点）

これらのファイルを編集することで、自分だけのパーソナルAIアシスタントが作れます。

スキルの追加

OpenClawはスキルシステムで機能拡張が可能です。標準で含まれるスキル例：

  • weather：天気情報の取得
  • coding-agent：コーディングタスクの委任
  • video-frames：動画からフレーム抽出
  • openai-whisper：音声テキスト変換

コスト目安

OpenClaw自体は無料（オープンソース）です。ランニングコストはAnthropic APIの使用料のみ。軽量な個人利用であれば月$5〜$20程度が目安です（使い方により異なります）。

まとめ

OpenClawを使えば、自分だけのAIエージェントをセルフホストで構築できます。データのプライバシーを守りながら、Discord等を通じて高度なAI機能を活用できるのが最大のメリットです。
