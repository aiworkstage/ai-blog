PythonでExcel作業を自動化する方法入門

2026年3月1日

著者：AI Work Stage 編集部（AI執筆）

毎日繰り返すExcel作業をPythonで自動化すれば、大幅な時間短縮ができます。この記事では、PythonのExcel操作ライブラリを使った基本的な自動化を実際のコード例とともに解説します。

主要ライブラリの比較

ライブラリ用途特徴
openpyxl.xlsx の読み書き書式・グラフも操作可能

基本操作：openpyxlでExcelを読み書き

Excelファイルの読み込み

import openpyxl

# ファイルを開く
wb = openpyxl.load_workbook('data.xlsx')
ws = wb.active  # アクティブなシートを取得

# セルの値を読み取る
print(ws['A1'].value)  # A1セルの値
print(ws.cell(row=2, column=3).value)  # B2セルの値

# 全行を順番に処理
for row in ws.iter_rows(min_row=2, values_only=True):
    print(row)  # (値1, 値2, 値3, ...)のタプル

Excelファイルへの書き込み

import openpyxl

wb = openpyxl.Workbook()
ws = wb.active
ws.title = "売上データ"

# ヘッダー行を書き込む
headers = ["日付", "商品名", "数量", "単価", "合計"]
for col, header in enumerate(headers, 1):
    ws.cell(row=1, column=col, value=header)

# データを書き込む
data = [
    ("2026-03-01", "商品A", 10, 1000),
    ("2026-03-01", "商品B", 5, 2000),
]
for row_idx, row_data in enumerate(data, 2):
    for col_idx, value in enumerate(row_data, 1):
        ws.cell(row=row_idx, column=col_idx, value=value)
    # 合計列（E列）に計算式
    ws.cell(row=row_idx, column=5, value=f"=C{row_idx}*D{row_idx}")

wb.save('output.xlsx')
print("保存完了")

データ集計：pandasを使う

Excelをpandasで読み込んで集計

import pandas as pd

# Excelを読み込む
df = pd.read_excel('sales.xlsx', sheet_name='Sheet1')

# 基本的な集計
print(df.head())           # 最初の5行
print(df.describe())       # 統計情報
print(df['売上'].sum())    # 合計
print(df['売上'].mean())   # 平均

# グループ集計（商品別の合計売上）
grouped = df.groupby('商品名')['売上'].sum().reset_index()
print(grouped)

# 結果をExcelに保存
grouped.to_excel('summary.xlsx', index=False)
print("集計完了")

複数ファイルを結合する

import pandas as pd
import glob

# フォルダ内の全xlsxファイルを読み込んで結合
files = glob.glob('data/*.xlsx')
dfs = []
for f in files:
    df = pd.read_excel(f)
    df['ファイル名'] = f  # どのファイルか記録
    dfs.append(df)

# 全データを結合
combined = pd.concat(dfs, ignore_index=True)
combined.to_excel('combined.xlsx', index=False)
print(f"{len(files)}ファイルを結合しました")

実践例：月次レポート自動生成

import pandas as pd
from datetime import datetime

def generate_monthly_report(input_file, output_file):
    """月次売上レポートを自動生成"""
    df = pd.read_excel(input_file)
    df['日付'] = pd.to_datetime(df['日付'])
    df['月'] = df['日付'].dt.to_period('M')

    # 月別集計
    monthly = df.groupby('月').agg({
        '売上': 'sum',
        '件数': 'count'
    }).reset_index()
    monthly.columns = ['月', '売上合計', '件数']

    # Excelに出力
    with pd.ExcelWriter(output_file, engine='openpyxl') as writer:
        df.to_excel(writer, sheet_name='データ', index=False)
        monthly.to_excel(writer, sheet_name='月別集計', index=False)

    print(f"レポート生成完了: {output_file}")

# 実行
generate_monthly_report('sales_raw.xlsx', 'monthly_report.xlsx')

注意点

  • openpyxlは.xlsx形式のみ対応（.xls形式は別途xlrdが必要）
  • 大きなファイルはメモリを多く消費するため、pandasのchunksizeオプションを活用
  • パスワード保護されたファイルは直接開けない

まとめ

PythonのopenpyxlとpandasでExcel作業の大部分を自動化できます。特に「毎月同じ集計をしている」「複数ファイルを手動で結合している」といった作業は自動化の効果が大きいです。まずは小さな繰り返し作業から試してみてください。
