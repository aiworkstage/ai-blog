Stripeで決済を導入する方法【個人・法人対応】

2026年3月1日

著者：AI Work Stage 編集部（AI執筆）

Stripeは世界で広く使われるオンライン決済プラットフォームです。初期費用なし・月額固定費なしで始められ、個人でも法人でも利用できます。この記事では、Stripe公式ドキュメントをもとに基本的な導入手順を解説します。

Stripeの料金体系（2026年3月時点）

Stripeの手数料（日本の場合）：

  • 国内カード：3.6%（Visa/Mastercard）
  • 海外カード：3.9%＋¥30
  • 月額固定費：なし
  • 振込手数料：250円/回

※手数料は変更される場合があります。Stripe公式の料金ページでご確認ください。

アカウント登録

  1. stripe.com/jpにアクセス
  2. メールアドレス・名前・パスワードを設定
  3. メール認証を完了
  4. 本人確認（個人：マイナンバーカード等、法人：登記情報等）
  5. 銀行口座を登録

⚠️ 本番決済（ライブモード）には審査があります。テストモードは審査前から利用可能です。

無料登録する →

プログラミング不要で決済リンクを作成できます：

  1. Stripeダッシュボードにログイン
  2. 「Payment Links」→「Create new link」
  3. 商品名・価格を設定
  4. 生成されたURLを顧客に共有

物理的なモノを売る場合も、デジタルコンテンツも、この方法が最も手軽です。

プログラマー向け：Stripe Checkout の実装

自分のWebサイトに決済フォームを組み込む場合はStripe Checkoutを使います。

必要なもの

  • Stripeアカウント（テスト用・本番用のAPIキーを取得）
  • サーバーサイドの環境（Node.js、Python等）

Node.jsでの実装例

// npm install stripe
const stripe = require('stripe')(process.env.STRIPE_SECRET_KEY);

// チェックアウトセッションの作成
const session = await stripe.checkout.sessions.create({
  payment_method_types: ['card'],
  line_items: [{
    price_data: {
      currency: 'jpy',
      product_data: { name: '商品名' },
      unit_amount: 1000,  // 1000円
    },
    quantity: 1,
  }],
  mode: 'payment',
  success_url: 'https://example.com/success',
  cancel_url: 'https://example.com/cancel',
});

// session.url にリダイレクト

Webhookの設定（重要）

決済完了後に自動処理（メール送信・DB更新等）をしたい場合はWebhookが必要です：

  1. StripeダッシュボードでWebhook URLを登録
  2. checkout.session.completed イベントをリッスン
  3. 署名検証でセキュリティを確保
// Webhookハンドラ（Express.js）
app.post('/webhook', express.raw({type: 'application/json'}), (req, res) => {
  const sig = req.headers['stripe-signature'];
  const event = stripe.webhooks.constructEvent(
    req.body, sig, process.env.WEBHOOK_SECRET
  );

  if (event.type === 'checkout.session.completed') {
    const session = event.data.object;
    // 決済完了後の処理
    console.log('支払い完了:', session.id);
  }

  res.json({received: true});
});

テスト方法

本番前にテストカードで動作確認できます：

  • テスト成功：4242 4242 4242 4242（有効期限：任意の未来日付、CVC：任意3桁）
  • テスト失敗（残高不足）：4000 0000 0000 9995

まとめ

Stripeは個人のデジタルコンテンツ販売から大規模ECまで対応できる柔軟な決済プラットフォームです。Payment Linksを使えばプログラミング不要で始められるため、まずは小規模からトライしてみることをお勧めします。
